### Backend Installation Manual
This project's backend uses a Python based web framework called Django. To get the backend server running please follow below steps.
1. Install Python (> 3.12). Can be downloaded from here: https://www.python.org/downloads/ 
2. Start a virtual environement. e.g `python3 -m venv .venv`
3. Activate virtual environment. e.g `source .venv/bin/activate`
4. Install packages specified in the requirements.txt file. e.g `pip install -r requirements.txt`
5. Start the backend server: `python manage.py runserver 0.0.0.0:8000`
6. Now you can access all the REST APIs offered by the backend here: http://127.0.0.1:8000/
7. The API documentation is automaticcally generated by Swagger Generator and it can be accessed by anyone here: http://127.0.0.1:8000/swagger/
8. Django automatically provides a powerful admin interface where admin users can manage data on. While it's not for building the entire front end around, it is a useful internal management tool for the organization. The admin site can here accessed here: http://127.0.0.1:8000/admin/
